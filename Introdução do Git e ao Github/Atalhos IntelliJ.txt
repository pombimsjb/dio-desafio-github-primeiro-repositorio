#Comandos IntelliJ

- Digitar "main" e dar TAB -> cria o método main
- Digitar "sout" e dar TAB -> traz a função System.out.println
- Atalho: ( CTRL + SHIFT + F10 ) -> Rodar a função sem efetuar a configuração na aba.
- Atalho: ( SHIFT + F10 ) -> Rodar uma função já configurada.
- Atalho: ( CTRL + SHIFT + / ) -> Comentar toda um bloco selecionado.
- Atalho: ( CTRL + / ) -> Comentar a linha.
- Atalho: ( CTRL + Y ) -> Apagar a linha.
- Atalho: ( CTRL + D ) -> Duplicar uma linha.
- Atalho: ( SHIFT + F6 ) -> REFATORAR Classe
- Atalho: ( SHIFT + F12 ) -> Ocultar barra lateral
- Atalho: ( CTRL + ALT + L ) -> Organizar o código
- Atalho: ( CTRL + SHIFT + Setinha acima ou abaixo ) -> Alterar posição da linha
- Atalho: ( CTRL + ALT + V ) -> Criar a variável.
- Atalho: ( CTRL + F4 ) -> Fechar aba
- Atalho: ( CTRL + N ) Busca por classes
- Atalho: ( CTRL + B) Entrar na classe selecionada
