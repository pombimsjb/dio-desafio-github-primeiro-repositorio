Os Comandos básicos para se usar no Git

rmdir -> Para remover diretórios
mkdir -> Para criar diretórios
ls -> para listar arquivos e diretórios na pasta
cd /e/ -> Navegar para o Disco ou pasta e/
del -> apagar arquivos



Comandos git

git add . -> Selecionar todos arquivos para commit
git commit -m "Mensagem" -> faz um commit com comentário
git push origin main -> envia o commit para o GitHub
git status -> puxar status dos arquivos locais
